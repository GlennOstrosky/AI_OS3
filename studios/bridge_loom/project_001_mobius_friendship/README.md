# Project 001 — Möbius Friendship Band

## Purpose
A simple, tangible symbol of unity-through-twist: one surface, one edge, one loop.

## Materials
- Paper strip (or ribbon), tape/glue, pen (optional)

## Build
1) Cut a strip.
2) Twist one end 180°.
3) Join ends to form the Möbius loop.
4) Optional: write a short blessing/phrase along the band.

## Meaning Layer
One path that returns transformed.
A bridge that crosses itself.
Three become one without ceasing to be three.
